Getting Started

Make your first request in minutes.

  1. Run the API hostdotnet run --project src/Gateway/Alphabet.AppWire
  2. Run this docs hostdotnet run --project src/Gateway/Alphabet.ApiDocs
  3. Confirm healthGET https://localhost:58441/health
  4. Load OpenAPIGET https://localhost:58441/openapi/v1.json

Authentication

Test protected endpoints with your own token.

Paste a JWT or API key once. Swagger UI, Scalar, Rapidoc, and the API console will reuse it for requests.
